Aplicando tags relacionadas à escrita

Podemos atribuir o significado de algo diretamente:

Na música temos um compositor chamado Chopin

Apliquei também a tag <mark> para marcar Chopin

Vou aplicar algumas outras marcações:

Como por exemplo o negrito usando a tag <strong>
Temos também o Itálico usando a tag <em>
Sublinhado usando a tag <u>
Indicar que o texto deve ser lido mas não considerado usando a tag <del>
Indicar um texto Inserido depois usamos a tag <ins>
Podemos deixar o texto grande usando a tag obsoleta <big> e deixar pequeno usando a tag <small>

Podemos escrever corretamente elementos matemáticos como:
202x10 usando a tag <sup> ou no caso do H2O usando a tag <sub>
